# nicovideo-next-video-canceler
[ニコニコ動画](https://www.nicovideo.jp/video_top)の動画再生終了後、次の動画へ遷移する前に自動的にキャンセルボタンをクリックするuserscriptです。
再生終了時に全画面で視聴していた場合、それも自動的に解除します。
